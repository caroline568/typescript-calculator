🧠 Overview

This project is a beginner-friendly calculator built using TypeScript, JavaScript, and HTML. It demonstrates how to create a simple interactive web application.

⚙️ Setup Instructions
Install TypeScript:
npm install -g typescript
Compile TypeScript:
tsc script.ts
Open index.html in browser
🧪 Minimal Working Example

The calculator performs:

Addition
Subtraction
Multiplication
Division

It also stores a history of calculations.
